**DevFlow Suite** is a zero-config VS Code extension for developers who want to stay organized without leaving their editor. Manage tasks, snapshot file states, and audit every workspace action — all stored locally with no backend required.

---

## Features

| | Feature | Description |
|---|---|---|
| 📋 | **Task Explorer** | Manage todos, priority tasks, and a recycle bin from a sidebar tree view |
| 🔍 | **Smart Scanning** | Auto-extracts `// TODO` and `// FIXME` comments into the task manager |
| 📌 | **Local Pin System** | Snapshot any file manually or on a timer; compare with Git-style diff |
| ⏱️ | **Activity Timeline** | Full audit log of every action — exportable as CSV, JSON, TXT, or PDF |
| ⭐ | **Tags & Stars** | Categorize and favorite pins for instant retrieval |
| 📝 | **Note Engine** | Attach markdown notes to tasks, stored locally |
| 🔒 | **100% Private** | No backend, no telemetry — all data stays on your machine |

---

## Getting Started

1. Install **DevFlow Suite** from the VS Code Marketplace
2. Click the **DevFlow icon** in the Activity Bar to open the workspace explorer
3. Open any file → click `📌` to pin its current state, or let auto-pin run on a timer

## Commands & Shortcuts

| Command | Shortcut (Win/Linux) | Description |
|---|---|---|
| Pin Current File | `Ctrl+Shift+P` × 2 | Snapshot the active file |
| View File Pins | `Ctrl+Shift+L` | Open the Pins panel |
| All Workspace Pins | Sidebar bookmark icon | View every pin across the project |
| Open Timeline | `Ctrl+Shift+T` | View the full activity log |
| Search Tasks | `Ctrl+Shift+F` | Search across all tasks |
| Scan / Refresh | `Ctrl+Shift+R` | Re-scan workspace for comments |

> Shortcuts can be remapped via **File → Preferences → Keyboard Shortcuts**
